Программа для тренировки слепой печати: показывает клавиши, которые вы нажимаете, автоматически меняет отображаемую раскладку вместе с Windows и не перехватывает мышку (можно кликать «сквозь» оверлей).

Что нового:
- При запуске открывается **окно настроек** (цвета/тема, отображаемые ряды клавиш, положение и прозрачность).
- Если закрыть окно настроек крестиком — программа **останется в трее**.
- В трее можно открыть настройки, скрыть/показать оверлей или **полностью выйти**.
