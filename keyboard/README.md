Программа подходящая для тех кто хочет научится слепой печати, будет показывать символы которые вы нажимаете, менять раскладку сразу с вами и легко поддаётся модификациям, есть понятные коментарии в коде и так же JSON файл где можно менять стиль программы за пару кликов. Программа не перехватывает мышку и можно спокойно нажимать на вещи под ней. Так же если пару секунд ничего не печатать то она станет прозрачной чтобы не мешать.
